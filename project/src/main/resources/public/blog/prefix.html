前言

古语有云，合抱之木,生于毫末;九层之台,起于累土;千里之行,始于足下，java发展了这么多年，框架纷繁复杂，但是万变不离其宗，想要练就绝世武功，一定不是学会多少花招，而是内功练好，java世界来讲，java se就是内功。

学习最快的方式肯定是看书，但是java的书籍有的很难入门，得看反复看三四遍才能都看懂，有的看一遍之后发现什么也没讲，还有一个问题是看完书之后如果不实践，容易理解的不深刻，很快就忘了。

我希望我能解决以上部分问题，这个《java从入门到放弃》系列博文是我根据《java编程思想》来编写的，主要是对原书的总结，同时加入了自己的理解，自己的一些经验和知识，加入了一些java 8新的概念，对原书的课后习题的解答等等，目的是为了巩固java基础，为后来学习框架打下基础
